一键脚本
---

一键脚本搭建 + 设置开启自启动

## 支持系统
CentOS 6+

Debian 7+

Ubuntu 12+

## 使用教程
1. 安装git
```shell
yum install git
```

2. git clone
``` shell
git clone -b master https://github.com/flyzy2005/ss-fly
```

中间换成自己fork的地址，原始地址已经被移除了

3. 密码+端口设置
```
ss-fly/ss-fly.sh -i password 1024
```
